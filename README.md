# Dotfiles

个人终端工具配置，一键部署：`bash install.sh`

## 工具清单

| 工具 | 用途 | 配置目录 |
|------|------|----------|
| Ghostty | 终端模拟器 | `ghostty/` |
| Starship | 命令提示符 | `starship/` |
| Yazi | 终端文件管理器 | `yazi/` |
| Neovim | 代码编辑器 | `nvim/` |
| zsh-autosuggestions | 历史记录自动补全 | `~/.zshrc` |
| zsh-syntax-highlighting | 命令语法高亮 | `~/.zshrc` |
| fzf | 模糊搜索（历史/文件） | `~/.zshrc` |

## 依赖安装

```bash
brew install ghostty starship yazi neovim zoxide fd poppler ripgrep \
  zsh-autosuggestions zsh-syntax-highlighting fzf
```

## Ghostty 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + T` | 新建 Tab |
| `Cmd + W` | 关闭 Tab |
| `Cmd + 1~9` | 切换到第 N 个 Tab |
| `Cmd + D` | 水平分屏（左右） |
| `Cmd + Shift + D` | 垂直分屏（上下） |
| `Cmd + [` / `Cmd + ]` | 切换分屏焦点 |
| `Cmd + Shift + ,` | 重载配置 |

## Yazi 快捷键

### 导航

| 快捷键 | 功能 |
|--------|------|
| `j` / `k` | 上下移动 |
| `l` / `Enter` | 进入目录 / 打开文件 |
| `h` | 返回上级目录 |
| `~` | 跳转到 Home |
| `.` | 显示/隐藏隐藏文件 |
| `/` | 搜索文件名 |
| `z` | zoxide 智能跳转 |
| `q` | 退出 |

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Space` | 选中/取消选中 |
| `y` | 复制 |
| `x` | 剪切 |
| `p` | 粘贴 |
| `d` | 移到回收站 |
| `D` | 永久删除 |
| `a` | 新建文件/目录（末尾加 `/` 建目录） |
| `r` | 重命名 |

### 标签页

| 快捷键 | 功能 |
|--------|------|
| `t` | 新建标签页 |
| `1~9` | 切换到对应标签页 |
| `[` / `]` | 上/下一个标签页 |
| `Ctrl + c` | 关闭当前标签页 |

## Neovim 快捷键

> Leader 键 = `空格`

### 基础操作

| 快捷键 | 功能 |
|--------|------|
| `jj` | 退出插入模式（代替 Esc） |
| `空格 w` | 保存 |
| `空格 q` | 退出 |
| `:qa` | 退出所有窗口 |

### 文件与窗口

| 快捷键 | 功能 |
|--------|------|
| `空格 n` | 打开/关闭文件树 |
| `Shift + L` | 下一个 buffer |
| `Shift + H` | 上一个 buffer |
| `空格 c` | 关闭当前 buffer |
| `Ctrl + h/j/k/l` | 在窗口间切换（左/下/上/右） |

### 搜索（Telescope）

| 快捷键 | 功能 |
|--------|------|
| `空格 ff` | 按文件名搜索 |
| `空格 fg` | 搜索文件内容（grep） |
| `空格 fb` | 搜索已打开的 buffer |
| `空格 fh` | 搜索帮助文档 |

### 代码跳转（LSP）

| 快捷键 | 功能 |
|--------|------|
| `gd` | 跳转到定义 |
| `gD` | 跳转到声明 |
| `gi` | 跳转到实现 |
| `gr` | 查找所有引用 |
| `gt` | 跳转到类型定义 |
| `Ctrl + o` | 跳回上一个位置 |
| `Ctrl + i` | 跳到下一个位置 |
| `空格 i` | 悬停信息（函数签名、文档） |
| `空格 rn` | 重命名 |
| `空格 ca` | 代码操作（快速修复） |
| `空格 f` | 格式化代码 |

### 代码结构（Treesitter）

| 快捷键 | 功能 |
|--------|------|
| `]f` / `[f` | 跳到下/上一个函数 |
| `]c` / `[c` | 跳到下/上一个类 |
| `vaf` | 选中整个函数 |
| `vif` | 选中函数体 |
| `vac` | 选中整个类 |
| `vaa` | 选中整个参数 |

### Git

| 快捷键 | 功能 |
|--------|------|
| `空格 dd` | 打开 diff 视图（未提交改动） |
| `空格 dh` | 当前文件的 git 历史 |
| `空格 da` | 整个仓库的 git 历史 |
| `空格 dc` | 关闭 diff 视图 |
| （自动） | 左侧 gitsigns 显示行变更状态 |

### 编辑

| 快捷键 | 功能 |
|--------|------|
| `J` / `K` | 快速上下移动 5 行 |
| `gcc` | 注释/取消注释当前行 |
| `gc` + 选区 | 注释/取消注释选中区域 |
| `空格 h` | 清除搜索高亮 |
| `<` / `>`（可视模式） | 缩进并保持选中 |
| `V` 选中后 `J`/`K` | 上下移动选中的行 |
| `空格 +` / `空格 -` | 文件树宽度 +5/-5 |

### 诊断

| 快捷键 | 功能 |
|--------|------|
| `[d` / `]d` | 上/下一个错误 |
| `空格 e` | 显示当前行错误详情 |
| `空格 dl` | 错误列表 |
